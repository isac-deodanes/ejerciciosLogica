7. En 1937, el matematico aleman Lothar Collatz, enuncio la conjetura de Collatz, tambien conocida como el problema de Ulam, conjetura 3n + 1, entre otros. Collatz enuncio que, a partir de cualquier numero natural, siempre se obtiene la unidad. Para ello se hace el siguiente procedimiento: 318 Estructuras de repeticion Tome un numero n y ejecute las siguientes operaciones: Si n es par, halle la division entera entre 2. Si n es impar, multiplıquelo por 3 y sumele 1. Con el resultado que obtenga, repita las operaciones anteriores, hasta obtener 1 como respuesta. Ejemplos: n = 13, se obtienen los siguientes resultados: 40, 20, 10, 5, 16, 8, 4, 2, 1. n = 6, se obtienen los siguientes resultados: 3, 10, 5, 16, 8, 4, 2, 1. Otra de las curiosidades de esta conjetura, es que cuando se llegue a 1 y se apliquen nuevamente las formulas, obtendra la secuencia 4, 2, 1 de forma infinita.
